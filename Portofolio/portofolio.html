About me

My name is Roland Kardouss, and I am originally from Damascus, Syria. I have a background in law, but after moving to Canada in 2017, I developed a strong interest in coding and technology. Since then, I have been dedicated to learning and growing my skills in software development, with a focus on building practical, and real-world applications. I have developed this passion by studying computer science at Lasalle College, where I gained solid foundational knowledge and practical experience in computer science. I developed an Apartment finder application aimed at helping users easily search for available rental apartments. The goal was to create a user-friendly interface that allows users to find listings, create account, and securely store their information. This project focuses on ensuring data safety, including usernames and passwords. Key Features: • Easy apartment search: Users can quickly find available apartments by filtering through various options. • User registration: Allows users to create accounts with secure username and password storage • Database security: Emphasized safe storage of user data and sensitive information, such as password Technologies Used: • Programming languages: C#, JavaScript, HTML/CSS. • Database: SQL Server Management • Security: Implemented hashing and encryption techniques to ensure password security
